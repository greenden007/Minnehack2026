# Minnehack 2026

## Overview

This is the repository for our Minnehack 2026 project. We are developing a 
mobile application designed to assist individuals with communication
difficulties during medical emergencies. The app allows users to create a 
profile with their medical information and emergency contacts, which can be
accessed by first responders in case of an emergency.

In the event of an emergency where the patient is nonresponive, first responders
can tap the constant app nitification on the phone screen (even though the phone
is locked) and it will ask whether this is an emergency. If the user confirms,
the app will automatically call 911 and display a pre-prepared text screen 
detailing the patient's medical situation.

## Features

- User Profile: Users can create a profile with their medical conditions, 
allergies, medications, long term diseases and emergency contact. Can also get
a doctor's aproval checkmark if the profile is verified by a doctor.

- Emergency Access: First responders can access the user's medical information
in case of an emergency, even if the phone is locked.

- Privacy Controls: Users can control what information is shared with first
responders and can choose to hide certain details if they wish.

- Emergency Call: In case of an emergency, the app can automatically call 911 
and display the user's medical information to assist first responders in 
providing appropriate care.
